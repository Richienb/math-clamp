# math-clamp

> [Clamp](https://en.wikipedia.org/wiki/Clamping_(graphics)) a number

## Install

```
$ npm install math-clamp
```

## Usage

```js
const mathClamp = require('math-clamp');

mathClamp(1, 2, 4);
//=> 2
```

## API

### mathClamp(number, min, max)
