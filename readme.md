# math-clamp

> [Clamp](https://en.wikipedia.org/wiki/Clamping_(graphics)) a number

## Install

```
$ npm install math-clamp
```

## Usage

```js
import mathClamp from 'math-clamp';

mathClamp(1, {minimum: 2, maximum: 4});
//=> 2
```

## API

### mathClamp(number, {minimum, maximum})
